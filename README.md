# 📈 NEPSE AI Stock Dashboard

An AI-powered stock analysis dashboard for Nepal Stock Exchange (NEPSE) built using Python and Streamlit.

## 🚀 Features

- Live market data integration (web scraping)
- Technical indicators (SMA, RSI)
- Buy / Sell / Hold signals
- ML-based prediction (Linear Regression, Random Forest)
- Confidence scoring system
- Market movers (Top Gainers, Losers, Opportunities)
- Alert engine for trading insights
- Interactive dashboard (Streamlit)
- AI-based query assistant

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- scikit-learn
- Plotly
- BeautifulSoup / Requests

## 📊 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
